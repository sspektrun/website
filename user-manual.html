Introduction

Overview

Spektrun is an Android application that allows the user to listen to any song, album, or playlist from the Spotify library without any interruption. That is to say, Spektrun has so far no promotional content of any kind, such as invasive ads or ads that pause the user's music listening.

Spektrun has several strong points, which make it a very robust application for the future. Currently, Spektrun has at least three strong points. Of which, we can highlight the following

Spektrun's Key features

  • Full Spotify Song's / Album library access
  • Shazam audio recognition
  • High security level-access
  • Youtube Video Player
  • Sync Song's lyrics
  • No ads

Installation & Setup

The installation of Spektrun is really simple. To do this, the user will have to go to the downloads page, select the version that the user needs most (it is advisable to download the latest version to avoid any kind of failure), wait for it to download, and once done, run the .apk package and install the application.

In case you didn't understand the above explanation, here are individual steps to make it easier for you to install Spektrun

Individual Steps

  1. Download the APK from the Downloads page.
  2. Allow installation from unknown sources in your settings.
  3. Open the APK file to install.

FAQ

If Spektrun doesn't open or ALWAYS crash at the start, maybe you're not meeting the requirements, please, check the requirements page before installing Spektrun

In the case that you fullfill the requirements, and the app keeps stopping / doesn't launch, maybe can be produced by an unknown error. In a near future, Spekrtun will have a bug-report section

Revisit the Downloads page and install the latest version. You may need to uninstall the old version first to avoid conflicts.

It's planned that in the future, Spektrun will warn the user when a new version comes out.

Spektrun it's a very very recent app, the app it's being developed at the moment, so, maybe, some aspects have not been fixed yet.

If your Spektrun app instance have crash, maybe has been due to your internet connection, or your vinculation with your own Spotify account. So, make sure to be always connected to Internet to avoid unknown errors, and also, make sure that you're login with your Spotify account

Theese well-known errors will be fixed in a near future.

At the moment, Spektrun's usage are only via invitation. In other words, if you're not in the whitelist you're not going to be able to use Spektrun.

In case you somehow access to the rest of the content from Spektrun, Spektrun has a banlist function that can ban a not-wished account from the app at any moment.

Due to Spektrun's logic and legal terms, Spektrun maybe it's not going to quit the whitelist at any moment.

So, if you're planning to use Spektrun, make sure that you're in the whitelist, because, in other words, the .apk package you'll install will be useless
