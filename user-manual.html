Introduction

Overview

Spektrun is an Android application that allows the user to listen to any song, album, or playlist from the Spotify library without any interruption. That is to say, Spektrun has so far no promotional content of any kind, such as invasive ads or ads that pause the user's music listening.

Spektrun has several strong points, which make it a very robust application for the future. Currently, Spektrun has at least three strong points. Of which, we can highlight the following

Spektrun's Key features

  • Full Spotify Song's / Album library access
  • Shazam audio recognition
  • High security level-access
  • Youtube Video Player
  • Sync Song's lyrics
  • No ads

Installation & Setup

The installation of Spektrun is really simple. To do this, the user will have to go to the downloads page, select the version that the user needs most (it is advisable to download the latest version to avoid any kind of failure), wait for it to download, and once done, run the .apk package and install the application.

In case you didn't understand the above explanation, here are individual steps to make it easier for you to install Spektrun

Individual Steps

  1. Download the APK from the Downloads page.
  2. Allow installation from unknown sources in your settings.
  3. Open the APK file to install.
